# Phase 25 Plan 1: SyncBatchNorm Header and Forward Pass

## Goal

Implement SyncBatchNorm header and forward pass (training and inference).

## Requirements

- DBN-01: SyncBatchNorm with all-reduce for mean/variance
- DBN-02: Cross-GPU batch statistics aggregation

## Deliverables

1. `include/cuda/neural/sync_batch_norm.h` - Public API

## Implementation Steps

### Step 1: Create Header (sync_batch_norm.h)

```cpp
namespace cuda::neural {

struct SyncBatchNormParams {
    int num_features;
    float eps = 1e-5f;
    float momentum = 0.1f;
    bool affine = true;
};

struct SyncBatchNormState {
    float* running_mean;
    float* running_var;
    float* gamma;  // scale
    float* beta;   // bias

    int num_features;
    bool training;
    float momentum;
    float eps;
};

// Forward pass for training
void sync_batch_norm_forward_training(
    const float* input,
    float* output,
    const float* gamma,
    const float* beta,
    SyncBatchNormState& state,
    int batch_size,
    int num_features,
    int spatial_size,
    float eps,
    float momentum,
    cudaStream_t stream = nullptr
);

// Forward pass for inference
void sync_batch_norm_forward_inference(
    const float* input,
    float* output,
    const float* gamma,
    const float* beta,
    const float* running_mean,
    const float* running_var,
    int batch_size,
    int num_features,
    int spatial_size,
    float eps,
    cudaStream_t stream = nullptr
);

} // namespace cuda::neural
```

### Step 2: Implement Forward Training

Algorithm:
1. Compute local mean: mean_local = sum(input) / (batch_size * spatial_size)
2. Compute local variance: var_local = sum((input - mean_local)²) / N
3. All-reduce mean across GPUs
4. All-reduce variance across GPUs (using synced mean)
5. Normalize: (input - mean) / sqrt(var + eps)
6. Scale and shift: gamma * normalized + beta
7. Update running stats: running_mean = (1-momentum) * running_mean + momentum * mean_sync

### Step 3: Implement Forward Inference

Simple normalization using pre-computed running statistics.

### Step 4: Handle Single-GPU Fallback

When device_count == 1, behave like regular batch norm (no all-reduce).
