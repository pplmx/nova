## Implementation Steps

### Step 1: Implement Backward Pass

Algorithm for backward pass:
1. Compute gradient w.r.t. normalized input: d_x_norm = d_y * gamma
2. Compute gradient w.r.t. variance: d_var = sum(d_x_norm * (input - mean)) * -0.5 * (var + eps)^(-3/2)
3. Compute gradient w.r.t. mean: d_mean = sum(d_x_norm) * -1 / sqrt(var + eps)
4. Compute gradient w.r.t. input: d_x = d_x_norm / sqrt(var + eps) + d_var * 2 * (input - mean) / N + d_mean / N
5. Compute gradient w.r.t. gamma: sum(d_y * normalized)
6. Compute gradient w.r.t. beta: sum(d_y)
7. All-reduce gradients across GPUs

### Step 2: Implement Running Stats Management

State structure:
```cpp
struct SyncBatchNormState {
    float* running_mean;  // size: num_features
    float* running_var;   // size: num_features
    float* saved_mean;    // for backward pass
    float* saved_var;     // for backward pass
    float* gamma;         // learnable scale
    float* beta;          // learnable bias

    bool training = true;
    float momentum;
    float eps;
    int num_features;
};
```

Initialization:
- running_mean = 0
- running_var = 1

Update during training:
```cpp
running_mean = (1 - momentum) * running_mean + momentum * batch_mean;
running_var = (1 - momentum) * running_var + momentum * batch_var;
```

### Step 3: Mode Switching

```cpp
void SyncBatchNormState::set_training(bool is_training) {
    training = is_training;
}
```
