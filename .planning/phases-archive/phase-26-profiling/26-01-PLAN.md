# Phase 26 Plan: Performance Profiling Implementation

## Goal

Implement performance profiling infrastructure with kernel timing and metrics collection.

## Requirements

- PROF-01: Kernel-level profiling with CUDA profiling tools integration
- PROF-02: Memory bandwidth and compute throughput metrics
- PROF-03: Collective operation latency tracking

## Implementation

### Step 1: Create Profiler Header

```cpp
namespace cuda::performance {

class ScopedTimer {
public:
    ScopedTimer(const char* name, Profiler& profiler);
    ~ScopedTimer();

private:
    const char* name_;
    Profiler& profiler_;
    cudaEvent_t start_, stop_;
};

class Profiler {
public:
    static Profiler& instance();

    void enable();
    void disable();
    bool is_enabled() const;

    void start_timer(const char* name);
    void stop_timer(const char* name);

    struct KernelMetrics {
        const char* name;
        float time_ms;
        size_t bytes_transferred;
        float bandwidth_gbps;
    };

    void record_kernel(const char* name, float time_ms);
    void record_memory_op(const char* name, size_t bytes, float time_ms);
    void record_collective(const char* name, const char* op_type, float time_ms);

    std::vector<KernelMetrics> get_kernel_metrics() const;
    void export_json(const std::string& filepath) const;
    void reset();
};

#define PROFILE_SCOPED(name) \
    cuda::performance::ScopedTimer scoped_timer_##__LINE__(name, cuda::performance::Profiler::instance())

#define PROFILE_KERNEL(name, ...) \
    do { \
        auto& profiler = cuda::performance::Profiler::instance(); \
        if (profiler.is_enabled()) { \
            cuda::performance::ScopedTimer timer(name, profiler); \
            __VA_ARGS__; \
        } else { \
            __VA_ARGS__; \
        } \
    } while(0)

} // namespace cuda::performance
```

### Step 2: Implement Profiler

Key components:
- Singleton pattern for global access
- CUDA events for timing
- Thread-safe metrics storage
- JSON export for analysis
