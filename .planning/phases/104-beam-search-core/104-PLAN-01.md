# Phase 104: Beam Search Core — GPU-accelerated beam search

## Goal

Users can perform GPU-accelerated beam search with memory-efficient KV management.

## must_haves

1. BeamSearchManager with configurable beam width (1-8)
2. Length-normalized scoring without underflow at 2000+ tokens
3. Reference-counted KV sharing across beams
4. Batch KV operations across multiple beams
5. TopK/TopP sampling with beam search
6. Beam search trace export for debugging

Create `include/cuda/inference/beam_search.h`:

```cpp
namespace cuda::inference {

struct BeamSearchConfig {
    int max_beams = 4;
    int min_beams = 1;
    float length_penalty = 0.7f;
    float temperature = 1.0f;
    int top_k = 1;
    float top_p = 1.0f;
    int max_length = 2048;
    bool enable_length_norm = true;
    bool enable_reuse = true;
};

struct BeamHypothesis {
    int64_t sequence_id;
    std::vector<int> tokens;
    float score;
    int length;
    bool finished;
};

class BeamSearchManager {
public:
    explicit BeamSearchManager(
        BlockManager* block_manager,
        const BeamSearchConfig& config
    );
    
    std::vector<BeamHypothesis> search(
        const memory::Buffer<float>& prompt_embeddings,
        int prompt_length,
        const stream::Stream& stream
    );
    
    void configure(const BeamSearchConfig& config);
    BeamSearchConfig get_config() const;
    
private:
    void initialize_beams(int prompt_length);
    void expand_beams(const memory::Buffer<float>& logits, const stream::Stream& stream);
    void prune_beams();
    void compute_length_normalized_scores();
    void export_trace();
    
    BlockManager* block_manager_;
    BeamSearchConfig config_;
    std::vector<BeamHypothesis> hypotheses_;
    std::vector<BeamHypothesis> next_hypotheses_;
    
    struct TraceEntry {
        int step;
        int beam_id;
        int token;
        float score;
        float normalized_score;
    };
    std::vector<TraceEntry> trace_;
};

} // namespace cuda::inference
```

Implement in `src/cuda/inference/beam_search.cpp`:
- Beam initialization with prompt tokens
- Logit expansion and scoring
- Length-normalized score computation
- Beam pruning with reference-counted KV sharing
- Trace export functionality
</action>

```cpp
class TopKSampler {
public:
    explicit TopKSampler(int k) : k_(k) {}
    
    std::vector<std::pair<int, float>> sample(
        const float* logits,
        int vocab_size,
        float temperature,
        uint64_t seed
    );
    
private:
    int k_;
};

class TopPSampler {
public:
    explicit TopPSampler(float p) : p_(p) {}
    
    int sample(
        const float* logits,
        int vocab_size,
        float temperature,
        uint64_t seed
    );
    
private:
    float p_;
};

} // namespace cuda::inference
```

Implement nucleus sampling (TopP) algorithm:
- Sort logits by probability
- Cumulative sum until reaching threshold p
- Sample from filtered set
