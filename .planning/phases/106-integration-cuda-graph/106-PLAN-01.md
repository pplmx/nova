# Phase 106: Integration & CUDA Graph — Combined features

## Goal

Users can combine beam search with speculative decoding and optimize all features with CUDA Graphs.

## must_haves

1. Beam search + speculative decoding combined
2. Dynamic block sizing (16/32/64 tokens)
3. Chunked prefill for long prompts (>16K tokens)
4. Persistent KV for CUDA Graph capture
