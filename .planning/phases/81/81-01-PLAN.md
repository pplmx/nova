## HYB Design (per D-01, D-02, D-03)

**Storage:**
- ELL portion: `values_ell` + `col_indices_ell` (max_nnz_ell rows × max_nnz_per_row)
- COO portion: `row_coo` + `col_coo` + `values_coo` (coordinate lists)
- Metadata: `row_to_format_` (0=ELL, 1=COO), `ell_row_count_`, `threshold_`

**Partition Algorithm:**
```
threshold = max_nnz_per_row / 2
For each row:
  - If row_nnz > threshold: route to ELL
  - Else: route to COO
```

**SpMV Algorithm:**
```
For ELL rows:
  For j in 0..max_nnz_per_row:
    y[row] += values_ell[row, j] * x[col_indices_ell[row, j]]

For COO rows:
  For k in 0..coo_nnz:
    y[row_coo[k]] += values_coo[k] * x[col_coo[k]]
```

// Sparse Matrix-Vector Multiply: y = A * x for HYB format
template<typename T>
void sparse_mv(const SparseMatrixHYB<T>& A, const T* x, T* y) {
    int num_rows = A.num_rows();
    
    // Initialize y to zeros
    for (int i = 0; i < num_rows; ++i) {
        y[i] = T{0};
    }

    // ELL portion SpMV
    int max_nnz = A.max_nnz_per_row();
    const T* ell_values = A.ell_values();
    const int* ell_col = A.ell_col_indices();
    
    for (int i = 0; i < A.ell_row_count(); ++i) {
        T sum = T{0};
        int base = i * max_nnz;
        for (int j = 0; j < max_nnz; ++j) {
            int col = ell_col[base + j];
            if (col >= 0) {
                sum += ell_values[base + j] * x[col];
            }
        }
        // Find the original row index for this ELL row
        // row_to_format tracks which rows are ELL
    }

    // COO portion SpMV (scatter-add)
    int coo_nnz = A.coo_nnz();
    const T* coo_values = A.coo_values();
    const int* coo_rows = A.coo_row_indices();
    const int* coo_cols = A.coo_col_indices();
    
    for (int i = 0; i < coo_nnz; ++i) {
        y[coo_rows[i]] += coo_values[i] * x[coo_cols[i]];
    }
}
