   - Protected helper: `dot_product()`, `axpby()`, `norm2()`

4. **ConjugateGradient<T>** (lines 200-350):
   - Require symmetric matrix (runtime check or static_assert)
   - Work vectors: r, p, Ap (3n storage)
   - Implement classic CG algorithm:
     ```
     r = b - Ax
     p = r
     while (!converged && iter < max_iter):
         Ap = A * p
         alpha = (r.r) / (p.Ap)
         x = x + alpha * p
         r_new = r - alpha * Ap
         beta = (r_new.r_new) / (r.r)
         p = r_new + beta * p
         r = r_new
         check convergence: ||r||/||b|| < tol
     ```
   - Return SolverResult with converged status

5. **GMRES<T>** (lines 400-600):
   - Constructor takes `restart` parameter (default 50)
   - Work vectors: Krylov basis V (m+1 vectors of size n), Hessenberg H
   - Full reorthogonalization for stability
   - Implement Arnoldi process:
     ```
     r0 = b - Ax
     beta = ||r0||
     v1 = r0 / beta
     for j in 1..m:
         w = A * vj
         for i in 1..j:
             hij = vi.w
             w = w - hij * vi
         h_{j+1,j} = ||w||
         if h_{j+1,j} == 0: break
         v_{j+1} = w / h_{j+1,j}
     ```
   - Solve least squares at each restart
   - Return best approximation

6. **BiCGSTAB<T>** (lines 650-850):
   - Work vectors: r, r_tilde, p, p_hat, s, t (6 vectors)
   - Implement van der Vorst algorithm:
     ```
     r = b - Ax, r_tilde = r
     p = r
     for iter:
         p_hat = M^-1 * p (identity for now)
         s = A * p_hat
         alpha = (r.r_tilde) / (s.r_tilde)
         x = x + alpha * p_hat
         s_vec = r - alpha * s
         if converged: break
         t = A * s_vec
         omega = (t.s_vec) / (t.t)
         x = x + omega * s_vec
         r = s_vec - omega * t
         if converged: break
         beta = alpha/omega * (r.r_tilde)/(s_vec.r_tilde)
         p = r + beta * (p - omega * s)
     ```

Include proper error handling and inline documentation.

Create roofline.hpp with the following components per D-06 through D-08:

1. **DevicePeaks struct** (lines 1-80):
   - `double fp64_peak_gflops` - theoretical FP64 peak
   - `double fp32_peak_gflops` - theoretical FP32 peak
   - `double fp16_peak_gflops` - theoretical FP16 peak
   - `double memory_bandwidth_gbps` - device memory bandwidth
   - `int compute_capability_major`, `compute_capability_minor`
   
   Peak FLOP/s computation:
   - FP64: 2 FMA/clock = 2 ops/clock per SM
   - FP32: 2 FMA/clock = 2 ops/clock per SM
   - Peak = num_SM * clock_MHz * ops_per_clock * 1e6 / 1e9 = GFLOPS
   - Use device props for clock rate (clockRate in kHz)

2. **get_device_peaks()** function (lines 100-150):
   - Query DeviceProperties
   - Lookup clock rate from device_prop.clockRate
   - Calculate theoretical peaks for each precision
   - Return DevicePeaks struct

3. **RooflineMetrics struct** (lines 200-250):
   - `double arithmetic_intensity` - FLOPs/byte
   - `double achieved_gflops` - measured performance
   - `double peak_gflops` - theoretical peak
   - `double achieved_bandwidth_gbps` - measured bandwidth
   - `double peak_bandwidth_gbps` - theoretical peak
   - `PerformanceBound bound` - enum: COMPUTE_BOUND, MEMORY_BOUND, BALANCED

4. **arithmetic_intensity()** template function (lines 300-350):
   ```cpp
   template<typename T>
   double arithmetic_intensity(long long flops, size_t bytes_accessed) {
       return static_cast<double>(flops) / static_cast<double>(bytes_accessed);
   }
   ```
   
   For SpMV: 
   - FLOPs = 2 * nnz (multiply-add per nonzero)
   - Bytes = nnz * sizeof(T) * 2 (read values + indices) + n * sizeof(T) (read x) + n * sizeof(T) (write y)
   - AI = 2*nnz / bytes

5. **RooflineAnalyzer class** (lines 400-550):
   - `void measure_kernel()` - warm-up + timed execution
   - `double compute_arithmetic_intensity(long long flops, const SparseMatrixCSR<T>& A)` - helper
   - `PerformanceBound classify(const RooflineMetrics& m)`:
     - If AI * peak_bandwidth > peak_compute: COMPUTE_BOUND
     - Else: MEMORY_BOUND
   - `double compute_peak_performance(Precision p)` - returns peak for precision

6. **Timing utilities** (lines 600-700):
   - `ScopedTimer` class using CUDA events
   - `measure_bandwidth()` helper for memory operations

7. **Roofline point calculation** (lines 750-850):
   - `RooflineMetrics analyze_kernel(const char* name, long long flops, double measured_time_ms, size_t bytes)`
   - Compute AI, achieved GFLOPS, compare to peaks
   - Return classification
